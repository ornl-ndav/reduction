The purpose of this manual is to briefly described the way NessiVector works.

Import NessiVector

From the Python interpreter, to be able to use the NessiVector, you must import the class called NessiVector from the module of the same name 

>>> from NessiVector import NessiVector

Create a NessiVector

  1. Create a float empty NessiVector

    2. Float is the default type of the NessiVector. To create such a vector, use the following rule

    >>> My_NessiVector_float = NessiVector()
    or 
    >>> My_NessiVector_float = NessiVector(type="float")
  2. Create a non-empty float NessiVector

    3. To create a NessiVector of type float that contains 5 elements initialized to 0

    >>> My_NessiVector_float_initialized = NessiVector(5)
    or 
    >>> My_NessiVector_float_initialized = NessiVector(5,"float")
  3. Create a double, int and unsigned int NessiVector

    4. Here are examples of how to create an empty NessiVector and 3 elements of type double. Use the same procedure to create NessiVector of types int and unsigned int. 

    >>> My_NessiVector_double = NessiVector(0,"double")
    or 
    >>> My_NessiVector_double = NessiVector(type="double")
    >>> My_NessiVector_double = NessiVector(3,"DOuble")
    or 
    >>> My_NessiVector_double = NessiVector(3,type="double")
  4. Orthographies accepted

    5. The different orthographies accepted are

    TypeAcceptedComments
    floatfloatcase insensitive
    doubledoublecase insensitive
    intintcase insensitive
    unsigned intuintcase insensitive
    unsigned intcase insensitive
    unsignedcase insensitive

Append values to the NessiVector

Once a NessiVector has been created, you can fill the vector with values

>>> My_NessiVector_Int.append(45)

>>> My_NessiVector_Int.append(35)

It also possible to append more than one value at a time as illustrated here

>>> My_NessiVector_Int.append(1,2,3,4,5,6)

    Doing simple operation between NessiVector

    1. Operators supported

      2. The operation supported between NessiVector are:

      - add: see section 3.6 of the DR_Lib_RS-1.doc
      - sub: see section 3.7 of the DR_Lib_RS-1.doc
      - div: see section 3.9 of the DR_Lib_RS-1.doc
      - mult: see section 3.8 of the DR_Lib_RS-1.doc
      - sumw: see section 3.10 of the DR_Lib_RS-1.doc

    2. Import operation

      3. In order to use these operations, you must first import them from the array_manip module: 

      >>> from array_manip import add   #to import add
>>> from array_manip import *     #to import all of them
    3. Procedure to use these operations

      1. Operator(NessiVector_A, NessiVector_B)

        2. For our illustration, we first need to create two 3 elements long NessiVector of type float and we will add them together using the operator add 
        >>> Vector_A = NessiVector()    #create a NessiVector of type float
        
>>> Vector_A.append(1.25)       #Vector_A[0] is now 1.25 
        
>>> Vector_A.append(2.35)       #Vector_A[1] is now 2.35
        
>>> Vector_A.append(3.45)       #Vector_A[2] is now 3.45
        
>>> Vector_B = NessiVector()    #same thing with Vector_B
        
>>> Vector_B.append(5.5,6.6,7.7)
        
>>> Vector_C = NessiVector(len(Vector_A),"float")   #define a 3 elements long NessiVector
        
>>> c = add(a,b)    # or c = a + b
      2. Operator(NessiVector_A, NessiVector_A_err2, NessiVector_B, NessiVector_B_err2)

        3. See previous section on how to create Vector_A_err2 and Vector_B_err2
        See previous section on how to define Vector_C_err2
        >>> Vector_C,Vector_C_err2 = add (Vector_A,Vector_A_err2,Vector_B,Vector_B_err2)
      3. NessiVector_result = NessiVector_1 operator NessiVector_2

        4. Simple operations between two NessiVector can be performed without using the array_manip module. These operations come free with the NessiVector module.
        OperationSymbols used
        Addition+
        Substraction-
        Multiplication*
        Division/

        >>> NessiVector_add = NessiVector_1 + NessiVector_2
        
>>> NessiVector_div = NessiVector_1 / NessiVector_2

    Methods

    These methods can be used by importing them from the NessiVectorUtils module.

    1. print_vect(NessiVector1)
      print_vect(NessiVector1,n)
      2.  print_vect(NessiVector_C)
      
5.7500000000000000 8.95000000000000000 11.150000000000000

      Displays the 10 first elements of the NessiVector called NessiVector_C. If the NessiVector contains more than 10 elements, the 10 first elements, 3 dots "..." and the last element of the NessiVector are displayed.

      The default number of elements displayed can be changed by giving this number as second variable to the print_vect method.

       print_vect(NessiVector_C,2)
      
5.7500000000000000 8.9500000000000000

      For float and double are displayed with 16 digits after the decimal separator.

    2. compare_vect(n,NessiVector1,NessiVector2)
      compare_vect(n,NessiVector1,NessiVector2,NessiVector3)

      3. Display side by side the first n elements of two, or three, NessiVectors.

    3. max_vect(NessiVector1)

      4. Return the maximum value of the NessiVector's elements

    4. min_vect(NessiVector1)

      5. Return the minimum value of the NessiVector's elements
