The purpose of this manual is to briefly described the way NessiList works.

Import NessiList

From the Python interpreter, to be able to use the NessiList, you must import the class called NessiList from the nessi_list module. 

>>> from nessi_list import NessiList

Create a NessiList

  1. Create a double empty NessiList

    2. Double is the default type of the NessiList. To create such a list, use the following rule

    >>> My_NessiList_double = NessiList()
    or 
    >>> My_NessiList_double = NessiList(type="double")
  2. Create a non-empty double NessiList

    3. To create a NessiList of type double that contains 5 elements initialized to 0

    >>> My_NessiList_double_initialized = NessiList(5)
    or 
    >>> My_NessiList_double_initialized = NessiList(length=5,"double")
  3. Create a int NessiList

    4. Here are examples of how to create an empty NessiList and 3 elements of type int. 

    >>> My_NessiList_int = NessiList(0,"int")
    or 
    >>> My_NessiList_int = NessiList(type="int")
    >>> My_NessiList_int = NessiList(length=3,"int")
    or 
    >>> My_NessiList_int = NessiList(3,type="int")
  4. Orthographies accepted

    5. The different orthographies accepted are

    TypeAcceptedComments
    doubledoublecase insensitive
    intintcase insensitive

Append values to the NessiList

Once a NessiList has been created, you can fill the list with values

>>> MyNessiListInt = NessiList(type="int")

>>> MyNessiListInt.append(45)

>>> MyNessiListInt.append(35)

>>> MyNessiListInt[1]

35

Extend values to the NessiList

It is also possible to extend a NessiList, which is appending more than one value at a time

>>> MyNessiListInt.extend([1,2,3,4,5,6,7])

    Doing simple operation between NessiList

    1. Operators supported

      2. The operation supported between 2 NessiLists or 1 NessiList and a constant are:

      - add: see section 3.6 of the DR_Lib_RS-1.doc
      - sub: see section 3.7 of the DR_Lib_RS-1.doc
      - div: see section 3.9 of the DR_Lib_RS-1.doc
      - mult: see section 3.8 of the DR_Lib_RS-1.doc
      - sumw: see section 3.10 of the DR_Lib_RS-1.doc

    2. Import operation

      3. In order to use these operations, you must first import them from the array_manip module: 

      >>> from array_manip import add   #to import add
>>> from array_manip import *     #to import all of them
    3. Procedure to use these operations

      1. Operator(NessiList_A, NessiList_B)

        2. For our illustration, we first need to create two 3 elements long NessiList of type double and we will add them together using the operator add 
        >>> List_A = NessiList()    #create a NessiList of type double
        
>>> List_A.append(1.25)       #List_A[0] is now 1.25 
        
>>> List_A.append(2.35)       #List_A[1] is now 2.35
        
>>> List_A.append(3.45)       #List_A[2] is now 3.45
        
>>> List_B = NessiList()    #same thing with List_B
        
>>> List_B.append(5.5,6.6,7.7)
        
>>> List_C = NessiList(len(List_A),"float")   #define a 3 elements long NessiList
        
>>> c = add(a,b)    # or c = a + b
      2. Operator(NessiList_A, NessiList_A_err2, NessiList_B, NessiList_B_err2)

        3. See previous section on how to create List_A_err2 and List_B_err2
        See previous section on how to define List_C_err2
        >>> List_C,List_C_err2 = add (List_A,List_A_err2,List_B,List_B_err2)
      3. NessiList_result = NessiList_1 operator NessiList_2

        4. Simple operations between two NessiList can be performed without using the array_manip module. These operations come free with the NessiList module.
        OperationSymbols used
        Addition+
        Substraction-
        Multiplication*
        Division/

        >>> NessiList_add = NessiList_1 + NessiList_2
        
>>> NessiList_div = NessiList_1 / NessiList_2
